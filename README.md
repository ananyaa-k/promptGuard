# PromptGuard

Built PromptGuard, an automated LLM red-teaming platform that evaluates AI safety and privacy risks across 50+ adversarial vectors. Engineered a 3-tier evaluation system (embedding cache → classifier → judge) that reduced evaluation costs by 97% while maintaining quality. The project required handling sensitive prompt data responsibly, implementing audit trails, and designing transparent evaluation workflows—skills directly applicable to privacy operations in SaaS environments where responsible data handling and compliance documentation are critical.

## What technologies are used for this project?

This project is built with:

## How can I run this project locally?

1. Clone the repository
2. Navigate to the project directory
3. Install the dependencies using `npm i`
4. Start the development server using `npm run dev`
